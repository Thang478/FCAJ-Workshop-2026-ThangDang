### [Blog 1 - OPTIMIZING COSTS AND SIMPLIFYING NETWORK ARCHITECTURE WITH "NATIVE ATTACHMENT" IN AWS NETWORK FIREWALL](3.1-Blog1/)
This blog introduces a solution utilizing the "Native Attachment" feature to directly attach the AWS Network Firewall to the Transit Gateway. This architecture eliminates the complexity of a traditional Inspection VPC, thereby optimizing resource costs and simplifying network traffic management.

### [Blog 2 - FLEXIBLE MANUFACTURING WITH AWS AND SOFTSERVE](3.2-Blog2/)
This blog presents a solution combining the AWS ecosystem (Generative AI, Serverless automation) and SoftServe to automate personalized manufacturing processes. Through this, manufacturers can easily deploy short-term customized batches with near 100% efficiency without the need for hardware reconfiguration.

### [Blog 3 - UNDERSTANDING AND DEPLOYING IAM ROLES INSTEAD OF ACCESS KEYS ON EC2](3.3-Blog3/)
This blog analyzes the reasons for and methods of deploying IAM Roles (Instance Profiles) instead of long-term Access Keys on EC2 instances, adhering to AWS security Best Practices. Combining AWS STS and IMDSv2 not only automates the management of temporary credentials but also strengthens the defense layer against SSRF attacks in real-world environments.
